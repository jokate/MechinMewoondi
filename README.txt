OOP FINAL Project.

Name : MEWOONDI.

Purpose.

1. Defense All Enemy Unit By Purchasing the Offensive Unit.

2. If the population of Enemy, You will be failed.

3. Purchase the Unit by 10 Coin.

4. Unit has fifth grades. ( Common, Rare, Unique, Legendary, Mythic )

5. Upgrade Unit to Clear the Boss.

This Project was implemented in C# class by using Unity Engine.

The Source code file is in Asset Folder

You can play game in MEWOONDI folder.

Implement the file oop.exe in MEWOONDI Folder.
